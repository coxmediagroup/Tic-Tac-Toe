Overview
==================

Implementation of Tic-Tac-Toe using Django and javascript.

Install
-------

Install with pip

:: pip install https://github.com/epicowl/Tic-Tac-Toe.git

Run local dev server

:: django-admin.py runserver

Navigate to localhost:8000/games/tic-tac-toe/
